# CleanCare (크린케어) 시스템 V3.0 기획서 - 모바일 네이티브 앱 (플랜 B)

본 기획서는 웹앱의 한계를 극복하고 디바이스 하드웨어를 직접 제어하는 네이티브 모바일 앱(Flutter)으로의 전환 및 고도화 아키텍처를 담고 있습니다.

## 1. 시스템 아키텍처 변경점 (Plan B)
기존 모바일 웹 브라우저 접속 방식에서 **Flutter 기반 전용 네이티브 앱** 설치 방식으로 완전 전환합니다.

| 구분 | V2.0 (기존) | V3.0 (Plan B - 신규 적용) |
| --- | --- | --- |
| **플랫폼** | 모바일 웹 브라우저 (Next.js PWA) | **Flutter 네이티브 앱 (iOS/Android 지원)** |
| **카메라 제어** | HTML5 `<input type="file">` | **ar_flutter_plugin 및 네이티브 Camera 하드웨어 직접 제어** |
| **로컬 저장소** | 웹 브라우저 IndexedDB (초기화 위험) | **Isar Database (초고속 네이티브 로컬 DB, 영구 보존)** |
| **위치 추적** | 브라우저 Geolocation API (오차 큼) | **네이티브 Geolocator API (최상위 정확도 GPS 칩 제어)** |

## 2. 모바일 앱 핵심 기능 5단계 완비 (구현 완료)

### 2.1 AR 공간 스캐닝 및 스팟(구역) 영구 기억
*   **AR 3D 맵핑:** 스마트폰 렌즈를 통해 실제 현장의 바닥과 벽면을 실시간 스캐닝.
*   **스팟 핀(Node) 고정:** 창문 1, 창문 2 등 비슷한 구조물을 구분하기 위해 AR 공간상에 직접 핀을 꽂고 이름을 명명.
*   **미니맵 2D 동기화:** 3D 공간에 꽂은 핀들이 우측 상단 2D 도면 미니맵에 자동 연동.

### 2.2 On-Device AI 렌즈 검증 (조작 원천 차단)
*   **초점/흔들림 감지:** 픽셀 분산(Laplacian Variance) 알고리즘으로 흔들린 사진 강제 컷.
*   **고스트 가이드 (Onion Skinning):** 전 사진 촬영본을 카메라 뷰포트에 35% 투명도(Cyan 톤)로 겹쳐 띄워 완벽한 구도 유도.
*   **화각/장소 일치율 매칭:** 전/중/후 사진의 엣지 픽셀을 비교하여 일치율 50% 미만 시 다른 장소로 간주하여 촬영 거부.

### 2.3 오프라인-퍼스트 (Offline-First) 무중단 동기화
*   **Isar DB 즉각 저장:** 통신 단절(지하실, 산간 오지) 상황에서도 로컬 DB에 사진과 메타데이터 최우선 암호화 저장.
*   **백그라운드 Sync:** 와이파이/데이터 복구 시 작업자가 개입하지 않아도 서버(Firebase)로 자동 일괄 업로드 진행.

### 2.4 조작 불가 절대 위변조 방지 워터마크
*   **픽셀 융합형 워터마크:** 메타데이터(EXIF) 조작을 막기 위해 이미지 픽셀 자체 하단부에 `위도/경도`, `촬영 시간`, `현장 주소`를 텍스트로 강제 합성(Stamp).

### 2.5 극한 환경 예외 처리 (Tolerance & Fallback)
*   **AR 드리프트 오차 허용:** 센서 튐 현상을 고려하여 3미터 이내 오차는 정상 허용.
*   **관리자 승인 강제 촬영:** 3미터 이상 오차 발생 시에도 작업자 육안으로 장소가 확실하다면 **[수동 강제 촬영]** 기능 제공. 단, 이 경우 시스템과 팩스 보고서에 예외 처리 붉은색 플래그 영구 기록.
*   **원클릭 팩스 자동화:** 모든 촬영 종료 후 관공서(교육청 등) 팩스 번호만 입력하면 전체 보고서(PDF) 자동 생성 및 모바일 팩스 API 전송.

### 2.6 회사 필수 행정 서류 중앙 관리 시스템 (PC ↔ 모바일 실시간 동기화)
*   **PC 관리자:** `설정 > 회사 서류 관리` 메뉴에서 사업자등록증, 사회적기업인증서, 자활기업인정서, 여성기업인증서, 직접생산 증명서, 통장사본, 중소기업확인서, 건물위생관리업영업필증, 저수조청소업영업필증, 소독방역업영업필증 등을 Firebase Storage에 업로드.
*   **모바일 작업자:** 현장에서 관공서/고객이 서류를 요구할 때 앱 내 [현장 즉시 팩스 발송] 화면을 열면 서버에 등록된 서류 리스트가 실시간 동기화되어 표시.
*   **즉시 다운로드 + 팩스:** 서류를 터치하면 스마트폰 로컬 메모리로 자동 다운로드 → 팩스 번호 입력 → 즉시 발송.
*   **문서 추가/삭제:** PC 관리자가 새 서류를 올리거나 삭제하면 모바일 앱에 즉시 반영.

## 3. PC 웹 대시보드 V3.0 고도화 (구현 완료)
*   **AR 스팟 뱃지:** 모바일에서 올라온 사진에 [📍 창문 1] 등 AR 스팟명이 표시됨.
*   **강제 촬영 경고등:** 수동 승인으로 촬영된 사진에 붉은색 깜빡이는 경고 뱃지 표시.
*   **팩스 서버 문서 연동:** 팩스 발송 화면에서 완료된 프로젝트 보고서를 서버에서 직접 불러와 첨부 가능.
*   **회사 서류 관리:** 설정 메뉴 하위에 서류 업로드/삭제/미리보기 전용 페이지 신설.

> **다음 과제:** 실제 팩스 API(팝빌 등) 연동, 안드로이드/iOS 빌드 배포, 실전 현장 테스트.
