PC 관리자 메뉴얼

웹 브라우저에서 접속하는 관리자 전용 시스템입니다

1로그인 / 회원가입

  • 🌐브라우저 주소창에 시스템 URL을 입력하거나 북마크를 클릭합니다.
  • 📧이메일 주소와 비밀번호를 입력 후 로그인 버튼을 누릅니다.
  • ✅최초 사용 시 회원가입을 클릭하여 관리자 계정을 생성합니다.
  • ✅로그인 후 자동으로 대시보드로 이동합니다.
  • ⚠️비밀번호는 6자 이상으로 설정해 주세요.
💡 Tip 브라우저의 "비밀번호 저장" 기능을 활용하면 다음 접속 시 자동 입력됩니다.

로그인 화면

2대시보드 — 통합 관제센터

  • 📊상단 3개 카드에서 긴급 조치 / 진행 중 / 완료 현장 수를 즉시 확인합니다.
  • 🚨긴급 조치 필요(빨간색) 현장은 가장 먼저 확인해야 합니다.
  • 🔍상단 검색창에서 현장명, 작업자명, 반려 사유로 빠르게 검색합니다.
  • 🗺️지도 관제 버튼을 클릭하면 카카오맵 위에서 현장 위치와 작업자 GPS를 실시간으로 봅니다.
  • 📋각 현장 행의 상세 보기 및 정산 버튼 → 사진·정산 정보를 확인합니다.
  • ✅완료 처리 버튼으로 작업을 완료 상태로 변경합니다.
  • ☀️우측 상단 날씨 위젯이 현재 기상 정보를 표시합니다.
⚠️ DB 전체 삭제 버튼은 모든 데이터를 영구 삭제합니다. 절대 실수로 누르지 않도록 주의하세요.

대시보드 화면

3작업 배정 (캘린더 + 지도)

  • 📅좌측 메뉴에서 작업 배정을 클릭합니다.
  • ➕우측 상단 + 새 작업 배정 버튼 → 작업 정보를 입력합니다.
  • 📝입력 항목: 현장명, 날짜, 담당 작업자, 주소, 담당자, 연락처, 사진 요구사항, 메모
  • 👤고객(Client)을 선택하면 주소가 자동 입력됩니다.
  • 🗺️지도 보기 탭 → 배정된 현장 위치를 카카오맵에서 확인합니다.
  • 📆캘린더에서 날짜를 클릭하면 해당일 작업 목록을 표시합니다.
  • 📅구글 캘린더 이메일이 등록된 작업자에게 자동 일정 초대가 발송됩니다.
💡 Tip 작업자 등록 시 구글 이메일을 입력해 두면 배정 즉시 구글 캘린더에 일정이 추가됩니다.

작업 배정 캘린더

4작업자 관리

  • 👥좌측 메뉴 작업자 관리 → 전체 현장 직원 목록을 관리합니다.
  • ➕+ 작업자 추가 → 이름, 직책, 전화번호, 구글 이메일, 상태를 입력합니다.
  • 🟢상태: 작업 중 / 대기(온라인) / 오프라인 3가지로 구분됩니다.
  • ✏️행 우측 수정 버튼으로 정보를 편집합니다.
  • 🗑️삭제 버튼은 확인 후 영구 삭제됩니다.
  • 📧구글 이메일 등록 시: 작업 배정 때 캘린더 초대 자동 발송

작업자 관리 화면

5사용자·정산·보고서·팩스·설정

👤 사용자(고객) 관리

고객사 정보(이름·주소·담당자·연락처)를 등록하면 작업 배정 시 자동 불러옵니다. 좌측 메뉴 사용자 관리에서 추가·수정·삭제합니다.

💰 결제·정산

현장 상세 팝업에서 금액·VAT 여부를 입력 → 세금계산서·명세서를 PDF로 출력합니다. 월별 매출 차트도 함께 확인됩니다.

📊 보고서

기간별 작업 현황·완료율·매출 통계를 조회합니다. 좌측 메뉴 보고서에서 접근합니다.

📠 팩스 발송

좌측 메뉴 팩스에서 고객사로 문서를 전송합니다. 수신 번호와 문서 내용을 입력 후 발송합니다.

1앱 설치

  • 🔗https://storage.googleapis.com/
    cleancare-cf307.firebasestorage.app/
    apk/cleancaresystem.apk
  • ⬇️APK 파일이 다운로드되면 열기를 탭합니다.
  • ⚠️출처를 알 수 없는 앱 설치 경고가 뜨면 설정 → 이 소스 허용을 활성화합니다.
  • ✅설치 버튼을 탭하면 홈 화면에 크린케어시스템 아이콘이 생깁니다.
💡 QR 코드로 설치 (더 쉬운 방법) 관리자가 qr.io에서 위 URL로 QR 코드를 만들어 인쇄해 두면, 갤럭시 카메라로 QR 스캔 → 바로 다운로드됩니다.

앱 설치 화면

2로그인 · 현장 목록

  • 🔐앱을 열면 로그인 화면이 나타납니다. 관리자에게 받은 계정으로 로그인합니다.
  • 📋로그인 후 오늘 배정된 현장 목록이 표시됩니다.
  • 📍현장 카드에는 현장명, 주소, 작업 상태가 표시됩니다.
  • ▶️현장 카드를 탭하면 작업 시작 화면으로 이동합니다.
  • 🗺️우측 하단 노란색 카카오 내비 버튼을 탭하면 현장까지 길 안내가 시작됩니다.

현장 목록 화면

31단계: 도면 촬영

  • 📸카메라가 자동으로 켜집니다. 작업장 도면(설계도)을 화면 안에 맞게 촬영합니다.
  • 🎙️음성 안내: "도면을 촬영해 주세요"
  • 🔘하단 큰 흰색 원형 버튼(셔터) 또는 볼륨 버튼을 눌러 촬영합니다.
  • ⏱️셔터 버튼을 누르면 3·2·1 카운트다운 후 자동 촬영됩니다.
  • ⚠️카운트다운 중 다시 볼륨 버튼을 누르면 취소됩니다.
  • 🔍촬영 후 AI 유사도 검사가 진행됩니다.
  • ✅사진이 마음에 들면 확인, 다시 찍으려면 재촬영을 탭합니다.
💡 볼륨 버튼 활용 (장갑 착용 시 권장) 볼륨 올리기 / 내리기 버튼 중 어느 쪽이든 셔터로 작동합니다. 장갑을 끼고도 쉽게 촬영할 수 있습니다.

도면 촬영 화면

4감독자 승인 시스템

⚡ 승인은 각 단계마다 필요합니다

도면 확인 → 장소 지정 → 작업 전 촬영 → 작업 중 촬영 → 최종 완료, 총 5회 감독자 승인이 필요합니다.

  • 🔢승인 화면에서 4자리 숫자 코드가 크게 표시됩니다.
  • 📲이 코드를 감독자(PC 관리자)에게 보여줍니다.
  • 💡감독자가 PC 화면에서 해당 코드를 입력하면 자동으로 승인됩니다.
  • 🌐인터넷 연결이 없을 경우: 감독자가 핸드폰 화면 입력창에 코드를 직접 입력합니다.
  • 🔅승인 대기 중에는 배터리 절약을 위해 화면이 자동으로 어두워집니다.
  • ✅승인 완료 시 진동 + 음성 "승인되었습니다"와 함께 다음 단계로 이동합니다.

승인 코드 화면

52단계: 촬영 장소 지정

  • 🎙️음성 안내: "촬영 장소를 지정하세요. 화면을 탭하면 마커가 추가됩니다"
  • 📍촬영된 도면 이미지 위에서 촬영할 장소를 손가락으로 탭합니다.
  • ✏️마커가 생기면 장소 이름을 입력합니다 (예: 주방, 화장실, 거실 등)
  • ➕추가하려면 다시 다른 위치를 탭합니다. 원하는 만큼 장소를 지정할 수 있습니다.
  • ✕잘못 지정한 마커는 탭 → 삭제 버튼으로 제거합니다.
  • ✅장소 지정 완료 후 장소 지정 완료 버튼을 탭합니다.
💡 Tip 작업 전·중·후 사진은 여기서 지정한 장소 수만큼 각각 촬영됩니다. 중요한 곳을 빠짐없이 지정하세요.

장소 지정 화면

63~5단계: 작업 전 · 중 · 후 사진 촬영

  • 🎙️음성 안내가 "1번 장소, 주방, 작업 전 촬영하세요"처럼 어느 곳을 찍어야 하는지 알려줍니다.
  • 👻작업 중·후 촬영 시 작업 전 사진이 반투명하게 오버랩되어 같은 위치에서 찍도록 도움을 줍니다.
  • 📊상단에 AI 유사도 점수가 표시됩니다 (동일한 장소인지 확인).
  • ⚠️AI가 유사도 낮음으로 판단하면 경고가 표시되지만, 재촬영 또는 그대로 진행을 선택할 수 있습니다.
  • 💧촬영된 사진에는 자동으로 날짜·시간·현장명 워터마크가 삽입됩니다.
  • ✅확인 → 다음 장소로 자동 이동합니다. 모든 장소 촬영 완료 시 승인 단계로 이동합니다.

작업 중 촬영 화면

7편의 기능 총정리

🔊 볼륨 버튼 셔터

볼륨 올리기·내리기 버튼 중 어느 쪽이든 셔터 역할을 합니다. 장갑을 끼고도 쉽게 촬영할 수 있습니다.

⏱️ 3초 카운트다운

셔터 버튼 클릭 → 3·2·1 카운트다운 후 자동 촬영. 카운트다운 중 볼륨 버튼을 누르면 취소됩니다.

🎙️ 한국어 TTS 안내

각 단계마다 음성으로 안내합니다. 우측 하단 🔊/🔇 버튼으로 음성을 켜거나 끌 수 있습니다.

📳 햅틱 피드백

촬영 시, 승인 완료 시, 단계 이동 시 진동으로 피드백을 제공합니다. 소리 없이도 확인 가능합니다.

🔅 화면 자동 감소

승인 대기 중 화면이 자동으로 어두워져 배터리를 절약합니다. 승인 후 자동 복구됩니다.

🗺️ 카카오 내비 버튼

우측 하단 노란 원형 버튼을 언제든 탭하면 현장까지 카카오 내비가 즉시 실행됩니다.

8작업 완료 · 오프라인 모드

  • 🎉모든 촬영·승인 완료 후 "수고하셨습니다! 모든 작업이 완료되었습니다" 화면이 표시됩니다.
  • ✅작업 완료 버튼을 탭하면 PC 대시보드에 완료 상태가 전송됩니다.
  • ⚠️작업 완료 전 앱을 나가려 하면 "작업이 완료되지 않았습니다" 경고가 표시됩니다.
  • 📶오프라인 모드: 인터넷이 없을 때 사진은 핸드폰에 저장됩니다.
  • ☁️인터넷 연결 복구 시 저장된 사진이 자동으로 서버에 업로드됩니다.
  • 🗂️사진은 월/일/현장명 폴더 구조로 정리되며, 3개월 후 자동 삭제됩니다.
